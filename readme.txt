Admin panel for connecting Google Business Profiles and managing reviews for Strix Google Reviews plugin.

== Description ==

The Strix Google Reviews Admin Panel is a companion plugin for the main Strix Google Reviews plugin. It provides a local admin interface for connecting Google Business Profiles, eliminating the need to use external services.

**Features:**

* **Admin Settings Panel**: Complete settings interface for developers
* **Google Maps API Integration**: Configurable API keys and testing
* **Local Google Business Profile connection interface**
* **Google Places API integration with autocomplete**
* **Comprehensive reviews dashboard with analytics**
* **Business profile overview with key metrics**
* **Review filtering and management tools**
* **Modern, responsive admin interface**
* **Real-time review statistics and trends**
* **Debug mode and logging capabilities**
* **Cache management for review data**
* **Seamless integration with main Strix Google Reviews plugin**
* **No dependency on external admin services**

== Installation ==

1. Make sure the main Strix Google Reviews plugin is installed and activated
2. Upload the `strix-google-reviews-admin` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. The admin panel will automatically integrate with the main plugin's interface

== Requirements ==

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Strix Google Reviews plugin (main plugin) must be installed and activated
* Google Places API key (configured in main plugin)

== Usage ==

1. Go to the Strix Google Reviews plugin settings
2. Navigate to the "Free Widget Configurator" tab
3. Click the "Connect" button for Google
4. A popup window will open with the local admin interface
5. Search for and select your Google Business Profile
6. Connect your profile to start retrieving reviews
